# Gemini Entry Point

Read and follow [`AGENTS.md`](AGENTS.md) before responding to a new project
request. Do not start with architecture questions or code.

Register the UiPath gate, verify the server-side Action Center decision, then
use `backend/phase0_interview.py` to guide the beginner one question at a time.

Mirror the user's language. If the project-start request is in English, keep
all approval guidance and Phase-0 questions in plain English.
